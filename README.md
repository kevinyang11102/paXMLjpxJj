# 物业管理系统 java221

## 项目概述


## 技术栈

### 后端技术

- Spring Boot
- MyBatis

### 前端技术

- HTML

## 运行环境

- 开发工具：IntelliJ IDEA
- 数据库：MySQL 5.7

## 部署步骤

1. 确保MySQL 5.7环境已经搭建完毕。
2. 使用IDEA或相应工具导入项目。
3. 配置数据库连接信息。
4. 直接运行Spring Boot应用。

## 功能模块

- 用户管理
- 房产信息管理
- 维修服务管理
- 费用收缴管理

## 系统角色

- 超级管理员：负责系统的维护和用户管理。
